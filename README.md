# AI Face Persona Overlay

This project uses Python and OpenCV to detect your face from the webcam and add cool persona overlays, colors, expressions, and particle effects in real time.

## How to Run
1. Install Python
2. Install required packages:
   pip install opencv-python numpy
3. Run the program:
   python persona_overlay.py

## Features
- Real-time face detection
- Detects eyes and smiles
- Shows emotions: HAPPY, SAD, CURIOUS, FOCUSED, NEUTRAL
- Shows different personas (Cyber, Nature, Fire, Ice, Void)
- Particle effects based on your expression
